Getting started

Clone the Repository

Inside your project folder, clone the repo.

If Https way :

$ git clone https://github.com/rssviana/rssv-workflow.git

Else if ssh way :

$ git clone git@github.com:rssviana/rssv-workflow.git

After clone the repo inside you project scope you need to enter inside rssv-flow.

$ cd rssv-workflow

Install node dependencies

The next step is install the node modules, all project dependencies.

$ npm install

Warning

Some peoples will need to add 'sudo' first and insert your password, don't worry everything is going to be ok. in that case please put this line code:

$ sudo npm install

Let's go

Almost ready, the final step is simple, to start the application and run the serve put this code:

$ npm start
